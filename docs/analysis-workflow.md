# Analysis Workflow

The notebook follows a clear analysis flow:

1. Load trade records and sentiment readings.
2. Clean and align dates across both datasets.
3. Map each trade to the matching daily sentiment regime.
4. Calculate PnL and performance summaries.
5. Compare long and short behavior across sentiment regimes.
6. Evaluate momentum versus contrarian patterns.
7. Review loss concentration and tail-risk behavior.
8. Summarize findings through charts and grouped tables.

This structure keeps the analysis reproducible and easier to audit.
