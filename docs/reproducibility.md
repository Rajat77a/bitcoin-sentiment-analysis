# Reproducibility

To reproduce the notebook results:

1. Place `historical_data.csv` and `fear_greed_index.csv` beside the notebook.
2. Open `bitcoin_sentiment_analysis.ipynb` in Jupyter Notebook, VS Code, or Google Colab.
3. Run the cells from top to bottom without skipping setup or cleaning cells.
4. Confirm that the date merge between trades and sentiment readings succeeds.
5. Re-run the summary and visualization cells after any dataset change.

If the CSV files are moved, update the notebook paths before running the analysis.
