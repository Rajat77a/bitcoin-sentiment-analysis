# Project Overview

This project compares crypto market sentiment with trader performance on Hyperliquid. The analysis joins trade-level data with daily Fear and Greed Index readings to study how trader outcomes change across market moods.

The main notebook focuses on practical questions:

- Do traders perform differently during fear, neutral, and greed regimes?
- Are long or short positions more effective under specific sentiment conditions?
- Do momentum or contrarian behaviors show stronger results?
- Where do losses concentrate, and which conditions create the highest tail risk?
